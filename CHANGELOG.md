## Unreleased

## v1.2.2 - 2026-07-13

## v1.2.1 - 2026-07-13

+ [修复] 移除重复写入的存储配置更新时间字段。

## v1.2.0 - 2026-07-12

+ [新增] 支持管理员在本地磁盘与 S3/MinIO 后端之间选择默认文件存储，文件和生成任务绑定具体后端并统一获取访问链接。
+ [调整] AI、存储与 Codex 配置改为显式统一保存，模型刷新和 S3 调试可使用未保存草稿，生图/视频工作台参数不再改写全局默认值。
+ [修复] 上传文件跨挂载目录时自动复制并清理临时文件，避免 `EXDEV` 导致上传失败。
+ [优化] 模型列表刷新失败时展示渠道连接等实际原因，便于排查配置与网络问题。
+ [调整] 生产 Docker Compose 改为拉取 GHCR `latest` 镜像，并在启动时强制检查更新，不再本地构建应用镜像。

## v1.1.0 - 2026-07-11

+ [调整] 推送到主分支后仅自动发布应用 Docker 镜像，不再创建 GitHub Release、文档镜像或 GitHub Pages 发布任务。

## v1.0.0 - 2026-07-11

+ [新增] 新增 `remote-docker` skill，可检查仓库状态、自动判断更新级别、升级版本并推送触发远程发布。
+ [新增] 新增服务端持久化方案文档和后端基础服务骨架。
+ [调整] 画布项目、素材库、文件访问和 AI 配置改为通过服务端 API 持久化，浏览器不再保存完整 AI API Key。
+ [调整] 图片、视频、音频和文本生成改为创建服务端异步任务并轮询结果，画布 loading 节点会记录 `generationTaskId` 以便恢复。
+ [调整] 生图工作台和视频创作台的生成记录改为读取服务端生成任务，删除记录只逻辑删除服务端任务。
+ [修复] 图片蒙版局部编辑改为通过后端异步任务传递 `mask`，不再被前端服务端任务路径拦截。
+ [调整] WebDAV 遗留同步域收窄为画布和素材，不再同步本地生成记录。
+ [优化] 本地 Agent 连接说明明确区分插件 / 手动 MCP 才会增加 Codex token 消耗，直接运行 Agent 不受影响。
+ [优化] 优化本地 Agent 连接说明，区分 Codex 插件启动和直接运行 Agent 两种方式。
+ [新增] Agent 对话消息改用 streamdown 流式渲染，提升长回复与 Markdown 内容展示效果。
+ [新增] Agent 新增画布、工作台、提示词库和素材等站点级工具，支持读取配置、触发生成与新增素材。
+ [新增] Agent 面板改为全站常驻右侧栏，开关时同步推动顶栏和页面内容，并保留独立入口按钮。
+ [新增] Agent 新增 `site_navigate` 工具，支持从对话中直接跳转首页、画布、工作台、素材和配置等页面。
+ [新增] Agent 对话运行中支持一键停止，发送按钮切换为红色停止按钮并中断当前 Codex turn。
+ [调整] Codex 会话改为站点级连续线程，不再按画布 ID 分隔，跨页面和跨画布保持同一上下文。
+ [调整] 移除仅前端调用 OpenAI responses 接口的在线画布助手旧逻辑，统一走 MCP + 本地 Codex 链路。
+ [新增] 画布节点支持统一维护名称字段，默认显示在节点上方，并可直接双击名称编辑。
+ [新增] 画布新增组节点，支持节点拖入/拆出分组、拖拽高亮吸附和移动组时带动子节点。
+ [调整] 画布节点顶部工具条改为点击选中节点后显示，避免鼠标经过节点时频繁弹出。
