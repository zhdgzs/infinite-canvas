## 风险点

- 成功、失败、取消响应必须与前端判别联合类型一致。
- 视频页面刷新后仍需通过 records 列表恢复运行中任务并继续轮询。
- 删除记录不能隐式取消任务，也不能删除生成文件。
- Provider 调试字段不得进入普通任务或记录响应。
